# 项目简介

Python深度学习酒店评论文本情感分析系统是一款基于深度学习的酒店评论文本情感分析工具，旨在帮助用户对酒店评价进行快速、准确的情感分析。

## 核心功能

- 登录注册：用户和管理员可通过账号密码登录系统。
- 酒店信息：系统提供酒店信息的录入和管理功能。
- 酒店数据可视化大屏：通过图表和图形展示酒店评价数据。
- 情感分析：对酒店评论文本进行情感分析，判断正面、负面或中性情感。
- 酒店信息预测：基于历史数据预测酒店未来的经营状况。
- 留言反馈：用户可以留下对酒店的反馈意见。
- 公告信息：系统管理员可以发布公告信息。
- 系统管理：管理员可以管理用户、酒店信息、系统设置等。
- 个人中心：用户可以查看个人资料、历史评价等信息。

## 适用场景

- 酒店行业：酒店管理者可以通过系统了解顾客对酒店的反馈，优化服务。
- 消费者：消费者可以查看其他顾客的评价，作为出行参考。
- 数据分析师：数据分析师可以利用系统对酒店评价数据进行深度分析。

## 技术特点

- 使用Python和Django框架进行开发，确保系统的稳定性和可扩展性。
- 基于MySQL数据库存储数据，保证数据的安全性和可靠性。
- 使用Vue.js实现前端界面，提供良好的用户体验。
